# Анализ A/B-теста таксопарка "Датапарк"

## Описание проекта

Таксопарк "Датапарк" провел продуктовый эксперимент по расширению радиуса поиска водителей:
- **Контрольная группа (0)**: радиус 3 км
- **Тестовая группа (1)**: радиус 5 км

**Гипотеза**: расширение радиуса повысит назначаемость заказов за счет большего числа доступных водителей.

**Риск**: слишком дальняя подача может ухудшить пользовательский опыт.

## Задачи

1. Проверка корректности сплит-системы

2. Выявление отложенных заказов

3. Оценка результатов эксперимента

4. Подготовка рекомендаций
